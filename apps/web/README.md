# TillLess Web (Frontend)

Next.js 14+ frontend application for TillLess shopping optimization platform.

## Stack
- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Forms:** React Hook Form
- **Auth:** BetterAuth

## Development

```bash
# Run dev server
nx serve web

# Build
nx build web

# Lint
nx lint web

# Test
nx test web
```

## Deployment
- **Platform:** Vercel
- **Auto-deploy:** Connected to GitHub (main branch)
